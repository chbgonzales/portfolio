Trinity-AMS

Trinity-AMS is an Asset Management System created using the LAMP (Laravel, Apache, MySql, PHP) Stack. It's the second major project that I've created during my Web Development bootcamp. It can create 3 kinds or users namely User, Technician and Admin which has different capabilities and access. It allows users to create, update and delete request for assets and asset repair/troubeshooting. Technician receives requests, takes ownership, process request, note actions made and make replacement request. Admin is the entity that accepts/rejects replacement requests and creates all types of users. This project was create to apply everything I've learned about LAMP stack and further grow my skills with utilizing it.
